# Chapter 3

<br>
塔斯顿转身走入花海，那些形态各异的花朵在他经过时微微摇曳。西尔诺急忙站起身，拍掉手上沾着的湿润泥土，跟上塔斯顿的步伐。

他想自己现在应该感到愤怒、仇恨、恐惧，至少也该有保持距离的警惕，可一切的情绪都被疑惑与对真相急迫的渴望压下了。他的大脑还在处理刚才那些信息，还有太多问题没有得到解答，他不知道塔斯顿是否愿意给他一个答案。

“你看起来很困惑，”塔斯顿说，“但你的问题远比我预想中来得更早，我以为要等到一切结束之后才会有人发现这里，或者再也不会有人看见这样的景色了。”

“这是哪里？”西尔诺谨慎地提问。

“灵魂的花圃。”塔斯顿在说话时依旧从容，仿佛他认为西尔诺知道这些也不会干扰到他想要做的事一般。事实上西尔诺现在确实不知道该怎么返回白厅，被突如其来的乱流裹挟着来到这里后，一切都超出了他的认知。

他猜测自己会感受到不同寻常的感觉并顺着方向来到这里，是因为他曾在缄默壁中接触过塔斯顿的灵，因此在灵魂脱离肉体之后，本能地对此感到了熟悉。

“灵魂的花圃？为什么叫这个名字？这里和树庭有什么关系？”西尔诺对塔斯顿与江文渚的布局有很多疑问，不过现在他想先尽量弄清楚现状。

“这里就是树庭，是你不曾来到的区域。”塔斯顿的脚步不疾不徐，靴底碾过潮湿的泥土与花瓣时发出细碎的声响，“你在这里看到了什么？”

西尔诺环顾四周，试着组织语言：“花，数不清的花。”

“它们不仅仅是花。”塔斯顿在一株深紫色的花前停下了脚步，那朵花的花瓣层层叠叠地向内卷曲，像一只紧握的手掌，他低头注视着花，目光里透出近乎悲悯的疏离。“每一朵花都代表着一个灵魂。”他说，“在它们根系间流淌的河你曾经见过，正是那条自树庭中央穿行而过的灵魂之河。”

西尔诺的呼吸停滞了一瞬，他再次看向那些花，这一次它们带给他的感觉截然不同了。“这些都是谁的灵魂？他们为什么会在这里？”他不禁发问。

塔斯顿没有回答，他折下了面前那朵深紫色的花，递向西尔诺。西尔诺愣了一下，伸手接过花朵，他看见茎脉的断面正飘散出无色的流体，下一秒他的眼前一花，一段陌生的记忆兀地灌入脑海。

……绵密的树林蔓延无尽，与他们相似的人们在树荫下欢笑，歌声婉转穿行于枝叶之间……黑色的潮水，所过之处树叶枯萎、花朵凋零，人们四散奔逃……世界崩塌，天空碎裂，黑潮化作深渊横亘大地，人们在潮水中挣扎，一个个凋落……

记忆到这里戛然而止，西尔诺松开手，深紫色的花朵落在地上，花瓣触及泥土的瞬间便迅速枯萎，化作一撮灰白的粉末被风吹散。他大口喘息着，那不属于他的恐惧仍残留在胸腔里，像是被冰冷的手指攥紧心脏。

“咳咳……那是什么？”他哑着嗓子问。

“初生种族。”塔斯顿答道，“灵言者所属的种族，比我们更早出现，是在这片大地上行走的第一个智慧种族，或许这就是为什么他们这样称呼自己。”

“什——”西尔诺彻底说不出话了，他的认知一次次被推翻，事到如今他对这些突然出现的爆炸性信息已经开始感到麻木。目光掠过无边的花海，那些花朵在他眼中忽然变得无比沉重，每一朵都是一条生命，一段被凝固的过去，“初生种族……他们……全都死了？”

“至少灵言者还活着。”塔斯顿望着那朵花消散的地方，神色平静得近乎冷酷，“也许他们是某种更接近于植物的生命形态，因此也更加强大与脆弱。他们曾创造了辉煌的文明，却没能抵御世界的剧变，污染如潮水般摧毁了他们赖以生存的根基，毁灭了他们的文明。或许在这之后还有离开树庭存活下来的幸运儿，这就不是我们能够知道的事了。”

这时西尔诺才想起来塔斯顿不知道沙法利的真实身份，他没有追查江文渚在离开他后做了什么，不然珐露那的房子里应该会留下有人进入过的痕迹才是。那么如果他说得没错，沙法利作为初生种族中的一员活过了这场灾难，污染也在他身上留下了不可磨灭的印记。

“所以灵言者把他们的灵魂保存在了这里？”西尔诺欲言又止，他看向那朵深紫色的花原本生长的地方，在被塔斯顿折断后只剩一片空空荡荡，“那这算是……杀死了他的灵魂吗？”

“他们的真身在泥土里，灵言者将他们藏起来了，没有人能找得到。你看见这朵花被折断，但过些时日它又会以另一朵花、另一种颜色重新开放。”

这一切都太超出常识，西尔诺几乎说不出任何可以作为回答的话来，就像他到现在也不明白为什么自己能站在这里与塔斯顿心平气和地交流。这实在是太荒谬了，所有的事都很荒谬，甚至让他怀疑自己是否在做梦，眼前的景象是白厅为他编造出的虚假记忆。

“你的目的到底是什么？你引发了战争，害死了那么多人，你到底想要从这里得到什么？”他急需一个话题将他拉回现实，所以他尖锐地提问了。

塔斯顿转过身来，他注视着西尔诺的目光里有种奇异的耐心，像是在为无知孩童耐心解释的大人，这让西尔诺感到不舒服，就好像对方从未将他摆上平等的位置看待。“你问我的目的，那我也问你一个问题。”他说，“你知道灵言者犯下的罪是什么吗？”

“……灵言者为了族人犯下了不可饶恕的罪，我们是从这份罪业中诞生的。”西尔诺确实不知道，所以他将江文渚与沙法利的对话复述了一遍，假装出一副自己知道的样子。

“你连这都不知道，却敢站在这里质问我？”塔斯顿轻轻笑了，“西尔诺，你比我想象的还要莽撞。”

西尔诺被这句话噎住了，他没想到塔斯顿一眼就看穿了自己。“那你告诉我。”他咬着牙用冒犯的语气试探，仿佛这样就能证明在这场谈话中他没有落于下风，“既然你知道，那就说清楚。你布下这么大一个局，死了那么多人，总该有个理由吧？”

“……是贪婪和傲慢。”塔斯顿没有再绕圈子，他微微摇了摇头，目光落在花丛之间，“这是他们所有人的罪，而灵言者是其中之最。又或许是她别无选择，然而别无选择依旧代表着做出了选择，改变不了本质。”

略顿了一下后，他继续缓步向前：“污染不是从外面来的，而是从内部，从他们自身。他们的能力与我们相似，都是通过操纵灵来达成的，这本该是纯净的力量，可当他们过度使用它去改变世界，世界也在反过来改变他们。他们的灵魂被侵蚀，越来越多的人像植物一样凋零死去，他们试图用能力抵御污染，而这反倒让它蔓延得更远，最终铸成了恐怖的灾祸。”

“假定你没有骗我，那你做的事跟这些有什么关系？”

“别急。”塔斯顿没有理会西尔诺的质问，他打断了西尔诺的话，继续按照自己的节奏不急不缓讲了下去，“为了阻止污染吞噬所有人，毁灭整个种族，他们将灵魂从肉体中剥离出来，封存在这里。他们认为只要灵魂还在，终有一天能找到净化污染的方法，使族人在一片干净的土地上重生。”

“结果是他们没能成功净化污染。”西尔诺说，不然这些属于初生种族的灵魂早就该活过来了，沙法利身上也不会出现黑斑。

“没有成功，但也没有失败。”塔斯顿触碰着身旁的花，指尖抚过湿润的土地，“他们的灵魂被埋进了泥土里，那么你看见的泥土里还有什么？”

西尔诺怔住了：“……灵魂之河。”

“这是属于我们次生种族的灵魂之河，是所有人死后灵魂汇聚之处，然而在千百年前并非如此。”塔斯顿偏头垂眸看向西尔诺，他的眼中不带任何情绪，让人完全看不出他的感情偏向，“曾经当一个人死去，他的灵魂会瓦解消散，成为天地间无主的灵中的一部分，这些灵在未来会组成另一个人新的灵魂。可这一切都被灵言者改变了，她将死后的灵魂汇聚成为河流，于是这条河流入树庭，流遍我们脚下的土地。”他微妙地停了一下，露出微笑，“你知道这意味着什么吗？”

“这……”西尔诺张开了嘴，他低头看着脚下那片湿润的泥土，清澈的水正从他的脚印边缘漫溢出来。“用灵魂之河……净化污染？”他难以置信地喃喃自语。

“用死者的灵魂冲洗初生种族的灵魂，于是携带着污浊的灵流回世间，重塑成被污染的人。待一个人死去，他的灵魂再次回归河流，于是污染便留在他的躯壳中，被遗弃在了大地上。直到千百年后污染遍布，唯独树庭成为净土，初生种族们将在此重生。”

如此可怕的话语就这样被塔斯顿轻声低语说了出来，西尔诺站在原地，感觉自己的血液在冷却，有什么冰冷的东西沿着脊椎往上爬。他忽然感到一阵强烈的恶心，不是身体上的，而是来自心理的抗拒。千百年来每一个死去的人，每一个在世上活过、爱过、恨过的人，他们的灵魂都没能真正安息，而是被汇聚起来灌入这片土地，去冲刷另一个种族留下的污染。

他想起了在他面前死去的人们，想起江文渚说过的话，想起那些他们始终没能找到答案的问题。现在答案就在他面前，赤裸而残酷。

“你很难接受这个事实，毕竟我们都曾将灵言者当作神明来崇拜。”塔斯顿语气淡然，却让西尔诺觉得更加刺耳，“这就是我们的神的真面目。”

“……不，你还没有回答我的问题。”西尔诺强迫自己抬起头，直视塔斯顿的眼睛，“你引发战争，操纵那么多人的命运，你到底是为了什么？你要把真相公之于众？还是说——”

“嘘——”塔斯顿比出了一个手势，“看来我们的谈话已经被树的主人听到了。”

西尔诺感到一阵风自身后吹起，吹乱了他额角的碎发，吹得花瓣纷纷扬扬飘起又洒落。他僵硬地顺着塔斯顿视线的方向转过身，只见花瓣纷扬旋舞间，一道洁白的身影已经赤足站立在花海之上。
